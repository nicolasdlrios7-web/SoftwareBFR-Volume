# MVP Scope

The first version proves one loop: **Create → Publish → Discover → Enter → Interact.**

## V1 user journey

1. User creates an account.
2. User opens Volume Studio.
3. User creates a Volume.
4. User adds spatial content (text, image, one 3D object).
5. User previews it.
6. User publishes it.
7. Volume receives a public URL.
8. Another user discovers it.
9. User enters the Volume.
10. User interacts with it.
11. User likes / saves / comments.

This is Vertical Slice 001 — see `docs/roadmap/PHASE_1_VERTICAL_SLICE.md`.

## Deliberately NOT built in early versions

- EEG / neural decoding, dream technology
- Full Flow OS / headset OS integration
- Advanced multiplayer, spatial voice chat
- Sophisticated recommendation AI
- Blockchain, creator monetization
- Procedural AI world generation, advanced avatar systems, live events
- Full game-engine capabilities

AI generation ("Create a forest at night" → scene) is a Studio enhancement, not a V1 requirement.

## Current phase

This repository is currently in **Phase 0 — Foundation** (see `docs/roadmap/PHASE_0_FOUNDATION.md`): no product features are implemented, only the architectural scaffolding that makes Phase 1+ safe to build in parallel.
