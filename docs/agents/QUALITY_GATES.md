# Definition of Done

A feature is not done because "it compiles." It is done when:

- [ ] Acceptance criteria satisfied
- [ ] Type checking passes
- [ ] Lint passes
- [ ] Unit tests pass
- [ ] Contract tests pass (when a shared schema is involved)
- [ ] Integration tests pass where the feature crosses layers
- [ ] No unauthorized architectural boundary crossed (see `docs/agents/OWNERSHIP.md`)
- [ ] Security implications considered for auth/uploads/user-content/moderation changes
- [ ] Documentation updated if behavior or a contract changed
- [ ] The actual user flow was verified, not just checks passing
- [ ] Diff reviewed
