# Phase 2 — Studio

## Goal

Build out Volume Studio beyond the Phase 1 minimal editor.

## Scope

- Scene hierarchy panel
- Properties panel
- Asset browser (`packages/storage`-backed uploads)
- Canvas with transform controls
- Preview mode
- Draft autosave, publish versioning UX
- `apps/worker` asset-processing jobs

## Deferred to later phases

AI-assisted scene generation, templates marketplace, full social product (Phase 4), rich runtime features — animations/audio/video/state machines/sequences (Phase 5), collaborative multiplayer (Phase 6), Multi Modal Dawn adapters (Phase 7).
